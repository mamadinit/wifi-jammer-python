# Jammer Wifi

Python WiFi DoS (Denial of Service) attack


# How to use

1. Install `python3` in your system.
2. Clone the project `https://github.com/Mr-Hax0r/wifi-jammer-python`

  ```bash
  cd wifi-jammer-python
  sudo python3 jammer_wifi.py
  ```
